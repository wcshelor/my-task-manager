# Reflection Domain

## Purpose

Support anti-spiral journaling and structured reflection. The app should help the user slow down and think, not pretend to be a therapist.

## Likely Object

- `ReflectionEntry`

Possible fields:

- date
- prompt type
- answers
- tags
- linked task or event, if useful
- optional follow-up action

## Possible Prompt Flow

1. What happened?
2. What am I feeling?
3. What story am I telling myself?
4. What facts do I actually know?
5. Is there a kinder or more realistic interpretation?
6. What is one small useful action?
7. What can wait?

## Interaction With Tasks / Planner

Reflection may produce a small useful action that becomes a task. It should not directly schedule time or write to Apple Calendar.

## Status

Scaffold only. No reflection mode exists yet.
